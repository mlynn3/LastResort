A simple, first time single player map featuring a retirement facility aesthetic. There's 45 enemies and it takes about 3-4 minutes long to complete.

For installing the map in eduke32 simply drag the .map and .ogg file into the same folder as the executable.

For installing the map in 20th Anniversary World Tour the Steam Workshop page is here: https://steamcommunity.com/sharedfiles/filedetails/?id=2297529953

The song included is Richard D. James's (a.k.a Aphex Twin, AFX) "4 Ny Groove" simply re-titled and reformatted for the sake of loading in the song. You can download the track here for free: https://archive.org/details/aphex_twin_user48736353001_afx/086+4+ny+groove.mp3
